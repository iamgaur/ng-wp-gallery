== Description ==

NG WP Gallery is a multiple layout image gallery plugin with creating responsive layout. This plugin provide photo galleries with beautiful fancybox. Its easy to use after upload all pictures you will get shortcode example [ngw_gallery id="136"].
This plugin include 3 main responsive image layout -

1) Masonry Layout
2) Masonry Gallery with onScroll Load
3) Grid Layout
